Text Summarizer

The purpose of this project is to familiarize myself with utilizing python's NLTK library, understand and develop API's, and learn how to communicate between client-side and server-side. This is a web application that utilizes python's Flask library and JavaScript's ajax call to send user inputted text on the client-side to the server-side where the summary is then generated (available in 5 different languages). I developed this project primarily with Python, JavaScript, HTML, & CSS.  Source Code 

Random Knights

Random Knights is a JavaScript-based 2 player game developed primarily utilizing HTML, CSS, & JavaScript. I made this game as a way to better grasp and understand the key concepts behind DOM Manipulation. After queuing in their own name, players will battle it out. However, theres a twist! Available abilities will be scaled according to the dice players roll & the Joker may appear and skip your turn!  Source Code 

Automated Birthday iMessage Bot

Developed utilizing Python's Py-iMessage Library, CronJob, & Excel, I made this project to make it easier to send "Happy Birthday" messages to your friends and families. After inputting the necessary information into the excel file, the code will routinely check for birthdays and send messages accordingly. Source Code
